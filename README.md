# Lionheart-BG---Map-Editor-
Terrain editor, based on the assets of the game Lionheart: Legacy of the Crusader - Black Isle Studios; Dedicated to creating maps for the Baldur's Gate saga.
Cache folder download > https://drive.google.com/file/d/1UzN_TL-HNgncYZ7H7IWiDISJIhE-dDFn/view?usp=sharing

HOW TO USE:

1-Download: Lionheart_Map_Editor.html
2-Download the cache folder from my Google Drive (all contents) [Cache](https://drive.google.com/file/d/1UzN_TL-HNgncYZ7H7IWiDISJIhE-dDFn/view?usp=sharing)
3-Unzip the cache folder to your preferred directory.
4-Open Lionheart_Map_Editor.html and load the cache folder. (A message will appear in the top center of your browser window)
5-The data will load and you'll be ready to use.
