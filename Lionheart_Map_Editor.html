Editor de mapas para Infinity Engine

Sigue estos pasos para cargar el contenido del juego:

  1. Descomprime el archivo Data.dat de tu directorio de instalación de Lionheart.
  2. Pulsa el botón de abajo y selecciona la carpeta Data que acabas de descomprimir (la que contiene Cache dentro).
  3. Puedes añadir tus propias carpetas con archivos .png personalizados dentro de esa carpeta Data; el editor también los cargará.

Requiere Chrome o Edge. La carpeta se lee solo en tu equipo; nada se sube a ningún sitio.
